## 📝 檔案命名規則

```
[學年度]-[系所簡稱]-[學制]-[類別].pdf
```

**學制代碼：**
- `四技日` = 四技日間部
- `四技進` = 四技進修部
- `二技日` = 二技日間部
- `二技進` = 二技進修部
- `二技在職` = 二技在職專班
- `碩士` = 碩士班
- `碩職` = 碩士在職專班
- `博士` = 博士班

**特殊類別：**
- `技優` = 技優領航專班
- `國際` = 國際學生專班
- `僑生專班` = 僑生專班
- `海青專班` = 海青專班
- `AI應用組` = 人工智慧應用組
- `智慧機械專班` = 智慧機械專班
- `輔系` = 輔系
- `輔系雙主修` = 輔系/雙主修課程

**範例：**
- `114-機械-四技日-課程科目表.pdf` (114學年度入學，機械工程系，四技日間部)
- `112-資工-四技日-AI應用組-課程科目表.pdf` (112學年度，資訊工程系，四技日間部 AI應用組)
- `113-環工-碩職-課程科目表.pdf` (113學年度，環境工程系，碩士在職專班)
